bootstrap

```
# install git, venv, and minimal system requirements
sudo apt-get install git python3-venv python3-dev libopenjp2-7-dev libtiff-dev libmpv-dev

# hardware configuration
Enable SPI via raspi-config (or add "dtparam=spi=on" to /boot/config)

# enable non-root user "derek" to use spi and gpio
sudo usermod -aG gpio derek
sudo usermod -aG spi derek
```
